Comment créer un site WordPress¶

Ce guide vous explique les étapes de base pour créer votre propre site WordPress.

1. Choisir un nom de domaine et un hébergement¶

La première étape consiste à choisir l'adresse de votre site (nom de domaine) et un endroit pour stocker vos fichiers (hébergement). Des hébergeurs populaires incluent : - Bluehost - OVH - SiteGround - Hostinger

2. Installer WordPress¶

La plupart des hébergeurs proposent une installation "en un clic".

Installation manuelle (si nécessaire) :¶

  1. Téléchargez WordPress sur wordpress.org.
  2. Décompressez le fichier.
  3. Envoyez les fichiers sur votre serveur via FTP.
  4. Créez une base de données MySQL.
  5. Renommez wp-config-sample.php en wp-config.php et ajoutez vos informations de base de données.
  6. Lancez l'installation en accédant à votre site.

3. Configuration de base¶

Une fois installé, accédez à votre-site.com/wp-admin pour vous connecter. - Allez dans Réglages > Général pour changer le titre du site. - Allez dans Réglages > Permaliens et choisissez "Titre de la publication" pour un meilleur référencement (SEO).

4. Choisir un thème¶

Allez dans Apparence > Thèmes et cliquez sur "Ajouter". Vous pouvez choisir parmi des milliers de thèmes gratuits ou en acheter un premium.

5. Ajouter du contenu¶

  • Articles : Pour les actualités ou le blog.
  • Pages : Pour le contenu statique (Accueil, À propos, Contact).

Félicitations, votre site WordPress est prêt !
